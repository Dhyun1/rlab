Algorithm

MLE를 계산하는 알고리즘을 먼저 생각해보면 다음과 같다.

  • Initialize \(\Psi^{(0)}\). for k = 1,2,…
  • Calculate \(L^{(k)}\) by (6) with \(\Psi^{(k-1)}\).
  • Calculate \(\Psi^{(k)}\) by (9) with \(L^{(k)}\).
  • Stop when it converges

강의록의 식 (6)은 식 (10)과 동치이며, 이는 \(\Psi^{-1/2}S_n\Psi^{-1/2}\)를 Eigen decomposition해서 구할 수 있음을 알 수 있다. (교재 p528-529의 내용)

\[ L(L^T\Psi^{-1}L+I_m)=S_n\Psi^{-1}L - (6) \] \[ diag(LL^T+\Psi)=diag(S_n) - (9) \] \[ (\Psi^{-1/2}S_n\Psi^{-1/2})(\Psi^{-1/2}L) = (\Psi^{-1/2}L)(I_m+ L^T\Psi^{-1}L) - (10) \]

(강의록의 식 (10)에 오타가 있습니다. 이게 맞는것 같습니다.)

식 (10)을 이용해서 L을 계산하는 방법을 조금 더 자세히 알아보면 주어진 \(\hat{\Psi}\)에 대해서, “uniqueness-rescaled” covariance \(S^*=\hat{\Psi}^{-1/2}S_n\hat{\Psi}^{-1/2}\)을 구한다.

이것의 Eigenvalue를 \(\hat{\Lambda}\), (Normalized) Eigenvector로 만든 행렬을 \(\hat{E}\)라 하면 다음과 같은 식이 성립한다. \[\hat{\Lambda} = I+\hat{\Delta}, \hspace{5mm} \hat{E}=\hat{\Psi}^{-1/2}\hat{L}\hat{\Delta}^{-1/2}\]

식 (10)을 잘 살펴보면, \(\Psi^{-1/2}L\)의 \(j\)번째 열이, \(I_m+ L^T\Psi^{-1}L\)의 \(j\)번째 값을 eigenvalue 로 갖는 (non-normalized) eigenvector의 모양임을 알 수 있다.

(\(I_m+ L^T\Psi^{-1}L\)는 diagonal이므로, 하나씩 비교해보면 된다. \(Av=\lambda v\)의 모양으로 만들 수 있음.)

이제, 식 10의 \(\Psi\)가 모두 추정된 \(\hat{\Psi}\)라고 생각하고, 대입해서 생각해보면, 행렬의 Eigenvalue는 당연히 같아져야 하므로,

\[I_m+L^T\hat{\Psi}^{-1}L = \hat{\Lambda}, \hspace{5mm} where \hspace{3mm} L^T\hat{\Psi}^{-1}L= \hat{\Delta}\]

이고,

\[L^T\hat{\Psi}^{-1}L= \hat{\Delta}\]

에 의해, Normalized eigenvector는

\[\hat{E}=\hat{\Psi}^{-1/2}\hat{L}\hat{\Delta}^{-1/2}\]

로 구할 수 있다.

이제 다시 MLE를 구하는 Algorithm으로 돌아가 생각해보면, 우리는 \(\hat{\Psi}\)를 알고 있는 상황에서, \(\hat{L}\)을 먼저 구해야하므로, 마지막 식

\[\hat{E}=\hat{\Psi}^{-1/2}\hat{L}\hat{\Delta}^{-1/2}\]

을 변형해 다음과 같은 식을 얻는다.

\[\hat{L}=\hat{\Psi}^{1/2}\hat{E}\hat{\Delta}^{1/2} = \hat{\Psi}^{1/2}\hat{E}(I_m+\hat{\Lambda})^{1/2} \]

이를 이용해서 다시 Algorithm을 정리하면,

  • Initialize \(\Psi^{(0)}\). for k = 1,2,…
  • Calculate Uniqueness-rescaled covariance \(S^{*(k-1)}=\Psi^{-1/2(k-1)}S_n\Psi^{-1/2(k-1)}\).
  • Get eigenvalues \(\hat{\Lambda}^{(k-1)}\) and eigenvectors \(\hat{E}^{(k-1)}\).
  • Calculate \(L^{(k)}\) by \(L^{(k)}=\Psi^{1/2(k-1)}\hat{E}^{(k-1)}(I_m+\hat{\Lambda}^{(k-1)})^{1/2}\)
  • Calculate \(\Psi^{(k)}\) by \(\Psi^{(k)}=diag(S_n-L^{(k)}{L^{(k)}}^T)\).
  • Stop when it converges

Result

자료를 입력한다. 실습시간에 했던 것처럼 교재 Table 8-5의 자료를 사용했다. 

data <-read.table("/Users/kyr0302/Desktop/Lab_4/T8-5.dat",header=FALSE,sep="");
n=nrow(data);p=ncol(data);
cov=cov(data); ## sample covariance

여기서 cov 함수로 구한 공분산은 Sample covariance인데, 이는 분산의 MLE와 다르다. \[ S_{sample}=\frac{1}{n-1}\sum_{k=1}^n(X_k-\bar{X})(X_k-\bar{X})^t \hspace{5mm} S_{MLE}=\frac{1}{n}\sum_{k=1}^n(X_k-\bar{X})(X_k-\bar{X})^t\]

따라서 우리의 MLE 계산식에 있는 \(S_n\)은, 항상 \(\frac{n-1}{n}\)를 곱하여 생각해주어야 한다.

계산하기 위해서, 먼저 Initial \(\Psi\)를 정한다. 여기서는 교재의 528p의 (9A-4) 에 나온 것을 생각하였다.

\[ \hat{\psi}_i=(1-\frac{1}{2}\dot\frac{m}{p})(\frac{1}{s^{ii}}) \].

여기서 \(s^{ii}\) 는 \(S^{-1}\)의 \(i\)번째 대각원소이다.

Factor의 개수는 2개로 하고 분석하도록 한다. (m은 조정할 수 있다.)

m=2;
psi <- (1-m/(2*p))*diag(1/diag(solve(cov)))

tol=10^(-5)             # For iteration
psi.old=diag(rep(0,5))  # For iteration
iter.num=1              # For iteration

이제 반복부분을 만들면,

while(norm(as.matrix(psi-psi.old),type="F")>tol){
  psi.old<-psi;
  S.star<-sqrt(solve(psi.old))%*%cov%*%sqrt(solve(psi.old))*(n-1)/n   ## Uniqueness-rescaled covariance 
  eig.res<-eigen(S.star);                                             ## Eigen decomposition
  E<-eig.res$vectors[,1:m]; Delta<-diag(eig.res$values[1:m]-1);       ## Ehat, Deltahat
  L<-sqrt(psi.old)%*%E%*%sqrt(Delta)                                  ## Calculate L
  psi<-diag(diag(cov*(n-1)/n-L%*%t(L)))                               ## Calculate Psi
  for(i in 1:p){
    if(diag(psi)[i]<0){
      diag(psi)[i]=0.005
    }                                                                 ## For Haywood case
  }
  iter.num<-iter.num+1
  # cat("iter.num=",iter.num,"psi=",psi[1,1],psi[2,2],psi[3,3],psi[4,4],psi[5,5],"\n")
  # Psi가 반복하면서 어떻게 변하는지 보고싶을 때 사용.
}
